# Retold Support

Use [Discord](https://discord.gg/S3g98zEY8a) for installation help, general support, discussion, and public playtesting.

Official downloads:

- [Modrinth](https://modrinth.com/mod/retold)
- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/retold)
- [GitHub Releases](https://github.com/xefensor/Retold/releases)

Confirm that Retold, Minecraft, NeoForge, and Java versions match the download page. Retold is currently required on both client and server.

Use the [bug form](https://github.com/xefensor/Retold/issues/new?template=bug-report.yml) for reproducible problems and the [suggestion form](https://github.com/xefensor/Retold/issues/new?template=suggestion.yml) for focused proposals.

Potential vulnerabilities should follow [`SECURITY.md`](SECURITY.md), not a public issue or Discord post.
